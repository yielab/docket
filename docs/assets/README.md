## Positioning rule for visuals

These assets must lead with **provisioning + isolation** and keep cost as a *guardrail*, not the
headline. When regenerating the hero GIF, do not make a
`docket cost` dollar table the centerpiece.

## Regenerating the hero GIF

```bash
python3 scripts/render-hero.py   # writes docs/assets/hero.gif (needs Pillow + FiraCode Nerd Font)
```

The renderer draws canned demo output ŌĆö no live OpenClaw install required. It uses only
anonymized agent names (webapp / api / blog); never put real client names or local paths in it.
Edit the `BLOCKS` list in the script to change the on-screen narrative.
